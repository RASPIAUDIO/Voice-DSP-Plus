# Raspberry Pi I2S 48 kHz AUTO

Advanced 48 kHz Raspberry Pi I2S image with automatic LINE/SQUARE geometry
detection at cold boot, two-channel capture/playback and DOA NeoPixel support.

The normal public installer intentionally selects the validated 16 kHz product
mode. This package is provided for developers who need 48 kHz I2S and are ready
to adapt the install service to the included SPI image.
